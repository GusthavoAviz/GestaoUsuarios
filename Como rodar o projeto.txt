1. Backend (API)
  Navegue até a pasta da API e execute:
   1 cd src/API
   2 dotnet run

2. Frontend
  Abra um novo terminal, navegue até a pasta do frontend e execute:
   1 cd frontend
   2 npm install  # Caso ainda não tenha instalado as dependências
   3 npm run dev
  O frontend ficará disponível em http://localhost:5173.
